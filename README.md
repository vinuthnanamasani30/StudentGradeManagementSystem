# Student Grade Management System

## Overview

A menu-driven C++ application to manage student records. The project demonstrates Object-Oriented Programming (OOP), STL vectors, and file handling by allowing users to store and manage student information.

## Features

- Add a student
- Display all students
- Search a student by ID
- Delete a student
- Calculate average marks
- Save records to a file
- Load records automatically when the program starts

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- STL (vector)
- File Handling

## How to Run

Compile the program:

```bash
g++ main.cpp -o student
```

Run the program:

```bash
./student
```

(On Windows, use `student.exe` or `.\student.exe`.)

## Sample Menu

```
1. Add Student
2. Display Students
3. Search Student
4. Delete Student
5. Average Marks
6. Exit
```
